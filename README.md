# StayGlow - Premium Airbnb Clone

## Features
- **Homepage**: Luxury search bar and featured stays.
- **Search System**: Filter by location and property types.
- **Listing Page**: Advanced sorting (Price/Rating) and filtering.
- **Property Details**: Real-time price calculation and amenity overview.
- **Booking System**: Securely stores reservations in the database.
- **Confirmation**: Animated success page for users.

## Technical Details
- **Language**: PHP (All pages)
- **Styling**: Internal CSS (Black & Glow Orange Theme)
- **Database**: MySQL (rsoa_rsoa285_34)
- **Interactions**: JavaScript for calculations and redirection.

## How to Setup
1. Import `database.sql` into your MySQL server.
2. Ensure `db_config.php` has the correct database credentials.
3. Open `index.php` in your web server.
